# Go_Project
> 这里存放学习Go语言产生的代码以及项目。

# Go 没有哪些内容
1. 没有类(class)
2. 没有方法重载
3. 没有继承
4. 没有枚举类型
5. 没有try/catch
6. 没有泛型

# Go 有哪些内容
1. struct 结构体 <- 没有类
2. 结构体的内嵌 <- 没有继承
3. 常量 <- 没有枚举类型
4. defer/recover/panic <- 没有 try/catch
5. 动态类型interface{} <- 没有泛型
6. 接口

# Go 特性
1. 头等函数，允许将函数分配给变量，作为其他函数的入参，返回值；函数可以作为对象，函数也可以是类型
2. 并行操作非常方便，通过go关键词启动协程
3. 通过通讯共享内存
4. 多路复用，channel的多路复用
